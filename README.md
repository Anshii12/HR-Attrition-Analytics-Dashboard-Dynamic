# HR Attrition Analytics Dashboard

## 📊 Project Overview
This project features an interactive **HR Attrition Dashboard** designed to identify and analyze turnover trends within a workforce of 40 employees. By leveraging advanced Excel functionalities, the dashboard transforms raw HR data into actionable insights, helping stakeholders understand "the why" behind employee departures across different demographics, departments, and salary bands.

## 🚀 Key Strategic Insights
Using the integrated **Master Insights** system developed for this project, the following critical trends were identified:
* **Mid-Career Flight Risk:** Employees in the **30–44 age bracket** represent the highest attrition volume (55% of total leavers), particularly among married male staff.
* **Departmental Leadership Gaps:** The **HR Department** shows a significant risk at the management level, with 66% of departures coming from HR Managers rather than entry-level staff.
* **Sales Compensation Lag:** 100% of leavers in the **Sales Department** were from the "Medium Salary" band and held Marketing degrees, suggesting a misalignment between market pay and job expectations.
* **R&D Expertise Drain:** Senior R&D staff in **High-Salary bands (Ages 45+)** are exiting at rates equal to junior staff, indicating that retention in this department requires non-monetary interventions.

## 🛠️ Technical Skill Set
* **Dynamic Data Modeling:** Linked all KPI metrics directly to **Pivot Tables** to ensure real-time accuracy.
* **Interactive Visualizations:** Implemented cross-functional **Slicers** (Gender, Marital Status, Department) with synchronized Report Connections.
* **Automated Narrative Generation:** Built a **Dynamic Master Insight** using complex Excel formulas (`IF`, `TEXT`, and `GETPIVOTDATA`) that automatically updates the "Insight" based on user filters.
* **Slicer-Aware KPIs:** Developed "Active Employee" and "Attrition Rate" cards using relational links to ensure data integrity during deep-dive analysis.

### How to Use the Dashboard
1. Download the `.xlsx` file from this repository.
2. Open in Microsoft Excel (Enable Macros/Content if prompted).
3. Use the **Slicers** on the right to filter by Gender, Marital Status, or Department.
4. Observe the **Insights** box to see the automated strategic summary for your selection.
